# PEF2-Projecte

Aquest projecte té com a objectiu desenvolupar un codi capaç de detectar la presència de fibril·lació auricular en una senyal obtinguda mitjançant BITalino.

## Informació del Projecte

Per a una documentació detallada, incloent-hi informació recopilada durant el desenvolupament, codis implementats, propostes futures i problemes pendents de solucionar, visiteu el nostre espai a Notion:

[PEF2 Electrocardiograma AF](https://trusted-dryosaurus-77d.notion.site/PEF2-Electrocardiograma-AF-1dac4d88f89680e1b386d75dc6d58df5?pvs=4)

## Estructura del Repositori

- `.idea/`: Configuracions específiques de l'entorn de desenvolupament.
- `data/`: Conjunt de dades utilitzades per entrenar i provar els models.
- `models/`: Models desenvolupats per a la detecció de fibril·lació auricular.
- `source/`: Codi font principal del projecte.
- `main.py`: Script principal per executar el projecte.
